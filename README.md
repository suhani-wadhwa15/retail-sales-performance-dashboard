# Retail Sales Performance Dashboard

## Overview

This project presents an interactive Retail Sales Performance Dashboard developed in Power BI to analyze sales performance across different regions, customer segments, payment modes, product categories, and shipping trends.

The dashboard enables business users to monitor key performance indicators and identify sales patterns through interactive visualizations.

---

## Tools Used

- Power BI
- Power Query
- DAX
- Data Modeling

---

## Key Performance Indicators (KPIs)

- Total Sales
- Total Profit
- Total Quantity

---

## Dashboard Features

- Sales analysis by Region
- Sales analysis by Customer Segment
- Sales analysis by Payment Mode
- Sales analysis by Category
- Sales analysis by Sub-Category
- Shipping Date Analysis
- Interactive filters for Country, State, and City

---

## Business Insights

- Compared sales across different customer segments.
- Identified top-performing product categories and sub-categories.
- Analyzed regional sales distribution.
- Evaluated customer payment preferences.
- Monitored shipping trends using interactive date filtering.
